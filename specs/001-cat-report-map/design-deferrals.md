# Design Deferrals: Catsy Design System Alignment

## Purpose

Track `DESIGN.md` visual opportunities intentionally deferred from the current implementation so quickstart remains a validation guide and design scope decisions stay explicit.

## Deferred Opportunities

| Opportunity | Reason Deferred | Follow-up Trigger |
|-------------|-----------------|-------------------|
| Nearby reports count with collar-tag motif | Not required for core report creation, monitoring, filtering, or current success criteria. | Add when report density or volunteer triage workflows need stronger summary cues. |

## Validation

Phase 7 catsy design alignment was validated against the quickstart design checks with desktop and mobile Playwright coverage for seeded map state, selected report detail, validation error, submit success, filter-empty state, reduced-motion preference, keyboard focus, and mobile stack order.
